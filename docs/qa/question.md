## Процесс реализации

### MVP

Минимально жизнеспособный продукт включает:

1. Telegram-бот с callback key
2. Поддержка ввода текста вручную
3. Поддержка PDF и TXT файлов
4. Анализ документов российской юрисдикции:

   - Пользовательское соглашение
   - Политика конфиденциальности
   - Оферта онлайн-школы / сервиса подписки
5. Базовая классификация рисков (🔴🟡🟢)
6. Бесплатный тариф (3 попытки анализа)
7. Транзитная обработка данных

### Вопросы к монетизации

- Ценовая стратегия для тарифов (Free, Pro, Pro+)
- Модель подписки vs оплата за анализ
- B2B API для образовательных платформ
- Партнерские программы

### Вопросы к реализации

- Выбор конкретного LLM (локальный vs облачный)
- Инфраструктура развертывания (cloud provider)
- Проведение тестирования
- Стратегия масштабирования
- Мониторинг и алертинг
- CI/CD процессы
